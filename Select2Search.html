Select2-like Library

Library JavaScript dengan Class untuk Select dengan Pencarian

Mendukung 2 mode: Data Langsung dan API dengan Limit & Search

Contoh Penggunaan

Info: API ini menggunakan JSONPlaceholder dengan limit 10 data. Pencarian akan memanggil API dengan parameter query.
Nilai yang Dipilih:
Belum ada yang dipilih

Implementasi Library

1. Library JavaScript (SelectSearch.js)

// Kode library akan muncul di sini

2. Inisialisasi dengan Data Langsung

// Inisialisasi dengan data langsung (in-memory) SelectSearch.init('countrySelect', { placeholder: 'Pilih negara...', searchable: true, multiple: true, data: [ {id: 'ID', text: 'Indonesia'}, {id: 'MY', text: 'Malaysia'}, {id: 'SG', text: 'Singapura'}, {id: 'TH', text: 'Thailand'}, {id: 'VN', text: 'Vietnam'}, {id: 'PH', text: 'Filipina'}, {id: 'US', text: 'Amerika Serikat'}, {id: 'GB', text: 'Inggris'}, {id: 'JP', text: 'Jepang'}, {id: 'KR', text: 'Korea Selatan'} ] });

3. Inisialisasi dengan API

// Inisialisasi dengan API SelectSearch.init('userSelect', { placeholder: 'Pilih pengguna...', searchable: true, multiple: false, api: { url: 'https://jsonplaceholder.typicode.com/users', limit: 10, mapResponse: function(response) { // Map response API ke format yang diharapkan return response.map(user => ({ id: user.id, text: user.name })); } } });

4. Inisialisasi dengan Opsi Minimal

// Inisialisasi dengan opsi minimal SelectSearch.init('productSelect', { placeholder: 'Pilih produk...', searchable: true, multiple: true });

5. Mendapatkan Nilai yang Dipilih

// Mendapatkan nilai yang dipilih const selectedValues = SelectSearch.getValue('productSelect'); console.log(selectedValues); // Atau dengan event listener SelectSearch.onChange('productSelect', function(values) { console.log('Nilai berubah:', values); });
